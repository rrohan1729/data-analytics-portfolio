# 🎬 Netflix — Improving Product Adoption Rate

## Overview
A product case study analyzing Netflix's challenge of low 
feature adoption among existing subscribers. Identified the 
core problem, conducted user research, and proposed BUZZ — 
a stories-based feature to improve product adoption rate 
among 24-40 age group in Tier 1 cities.

## Type
Product Management Case Study

## Tools & Methods
User Research | RICE Framework | Wireframing | KPI Design

## Key Findings
- 78% of users don't recall latest features launched by Netflix
- Only 37% satisfied with current feature guidance
- 50% of users are mobile-first
- 72% of paying customers come from Tier 1 cities

## Solution
BUZZ — A stories-based feature similar to Instagram Stories 
embedded in the Netflix app to communicate new features 
to existing users.
